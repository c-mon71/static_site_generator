Created as a project on boot.dev platform enables you to generate a full static website based on the markup files.

1. Create your content in the _/Content_ subfolder.
2. Use build.sh from the root folder to create html docs in the _/docs_ folder (not public since the default publishing platform is GitHub pages).
3. To create local instance use **main.sh** script from the root folder.
4. Change _/docs_ to _/public_ for other publishing methods.
